메모리: 프로그래밍을 하는데에 있어서 가장 중요한 컴퓨터의 하드웨어 중 하나
CPU: 연산을 하는데 사용 - 실제 계산을 하는 영역
Memory: ram이라고 부르는 영역을 담당 - 데이터를 저장하는 공간

메모리 셀
- 메모리를 구성하는 단위(1byte)
- 각각의 메모리 셀마다 주소가 존재한다.

1GB = 1,073,741,824 bit -> 주소: 0x000000 ~ 0x40000000(0x: 16진수라는 뜻)

변수를 사용하는 이유?
>> 직접적으로 메모리공간을 엑세스(접근)를 하면 지우지 말아야 하는 데이터를 지울 수도 있기 때문에
   우리가 사용할 메무리 공간에 대한 이름(변수)을 선언하는 것
>> 우리가 프로그램을 작성한 안에서는 실제로 만들어낸 변수들만 접근이 가능하기 때문에 다른 곳에서 사용하는 메모리 주소에 마음대로 접근하지 못한다.
